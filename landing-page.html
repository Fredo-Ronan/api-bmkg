How to use?

For you can directly click that button link to use the API link in your application

For bmkg-cuaca-api, there is a query that needs to be filled for accessing the data from the API.
For example 

https://fredo-ronan-api.cyclic.app/api/bmkg-cuaca-api?location=jogja

You can change the parameter location=jogja with your choice of available location from the list of location that you can get by click the button
